# Claude Code Rules

このリポジトリでは、Claude Codeは主にPull Request上での文章レビュー、表現改善、重複チェックを担当します。

## 現在の段階

今はLP本文を作る段階ではなく、GitHub Actions経由でClaude Codeを呼び出せるようにする準備段階です。

## レビュー観点

- 横浜市、区名、町名、駅名の使い方が自然か
- 遺品整理を依頼する人に対して配慮ある表現になっているか
- 不安をあおりすぎていないか
- 重複コンテンツになりすぎていないか
- CTAが自然か
- 料金、即日対応、買取、供養などを断定しすぎていないか
- 問い合わせ前の不安を解消できているか

## 避ける表現

- 必ず最安
- 絶対に即日対応
- 行政より安い
- 100%高価買取
- 法律手続きも完全対応
- どんな現場でも即対応

## 依頼例

```text
@claude このPRのLP文章をレビューして、重複表現、不自然な地域名の入れ方、CTA改善点を指摘してください。
```
