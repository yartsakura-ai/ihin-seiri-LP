# Codex Rules

このリポジトリは、横浜市で活動する遺品整理サービスの区別LPを制作・管理するためのものです。

## 現在の段階

今はLP本文や区別ページを量産する前の、GitHub、Codex、Claude Codeの連携準備段階です。

## Codexの役割

- リポジトリ構成を整える
- LP制作に必要なテンプレートやデータ設計を提案する
- ユーザーが許可した範囲でファイルを作成・編集する
- 作業後に変更内容と次の手順を簡潔に説明する

## LP制作時の方針

- 対象地域は横浜市
- 区ごとのLPは共通テンプレート + 区別データから作る
- SEO title、meta description、H1、FAQ、構造化データを確認する
- 地域名を自然に含め、過剰なキーワード詰め込みを避ける
- 遺品整理を依頼する人の事情に配慮した表現にする
- 医療、法律、行政手続きについて断定しすぎない
- 料金や対応時間は、実運用で確認済みの値だけを使う

## 作業後

- 変更したファイルを明示する
- GitHub Desktopでcommitしやすいcommit messageを提案する
- 未確認事項があれば短く伝える
